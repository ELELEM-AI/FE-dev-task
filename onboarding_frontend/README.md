# Prototype
- The prototype of this app is built using [Base44](https://base44.com/), a low-code platform for building web applications quickly.
- Tom Salvat (CEO, elelem.ai) is directly developing the frontend POC.
- The base44 app is synced to a [GitHub repository](https://github.com/base44dev/elelem-ai-bb43017c) for version control and collaboration.

## Running the app
```bash
npm install
npm run dev
```

## Building the app
```bash
npm run build
```
